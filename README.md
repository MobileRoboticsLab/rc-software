# Mobile Robotics Lab RC Software

## Installation

### 1. Flash SD with Ubuntu 22.04 Server 64-bit

```
login: ubuntu
password: ubuntu
```

### 2. Install Software

```
sudo apt update
sudo apt install -y git
git clone https://github.com/MobileRoboticsLab/rc-software
cd rc-software 
sudo ./setup_host.sh <RC#>
```

### 3. Reboot
